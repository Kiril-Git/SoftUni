SET A PROJECT:

- New Project
- Create POM

	В POM-file имаме:
*	 kонфигурацията на нашият проект
*	как се казва
*	коя JAVA използва
*	какви са неговите зависимости
*	как да бъде изграден до някакъв финален file

- Hibernate Configuration
- Persistence XML

*	Persistence XML е конкретна конфигурация на една конкретна библиотека, която сме добавли в рамките на РОМ
	Persistence служи за да окажем ма javax индиректно на hibernate как да се свърже с нашата база
	Persistence file  е нещо отделно, тъй като можем да подадем различен конфигурационен file, когато изграждаме нашият
	проект за production, да кажем.

!!!	РОМ-XML e maven конфигурацията на целия проект
	Persistance XML e maven конфигурацията на една конкретна библиотека
